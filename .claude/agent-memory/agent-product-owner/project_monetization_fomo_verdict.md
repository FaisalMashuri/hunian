Synthesizer menghasilkan verdict untuk model monetisasi baru ala "Fomo" (kontribusi data sewa anonim untuk unlock market intelligence agregat). PO gate: PENDING CONFIRMATION.

**Why:** Faisal mengusulkan model ini sebagai alternatif monetisasi. Synthesizer confidence MEDIUM — semua angka WTP/completion/densitas masih asumsi. Ini sesi PLANNING, bukan sprint commitment. Phase 0 gate tetap berlaku untuk eksekusi.

**Keputusan yang sudah SETTLED (tidak perlu konfirmasi ulang):**
- Fomo sebagai BILLING MODEL (subscription Rp50rb/bulan): DITOLAK — mechanics salah (frekuensi pakai rendah), bukan soal harga. One-time per-cycle tetap.
- Fomo sebagai DATA-COLLECTION MECHANIC (contribute-to-unlock): DIAMBIL — non-extractive, reciprocity, network effect nyata.
- Intel dasar (RANGE harga area) = fitur GRATIS pendorong completion; deeper intel = unlock dalam per-cycle flow yang sudah ada, BUKAN SKU/subscription baru.
- Cold-start: 5-10 kecamatan padat, hard threshold N>=30, tampilkan RANGE bukan angka tunggal, data fully anonymized AT WRITE TIME.
- Garis merah TAK BOLEH: jual agregat ke pihak ketiga (landlord/broker/developer) walau anonim; tampilkan agregat N<30 tanpa label; link kontribusi ke profil user.
- Fasing SEQUENTIAL: Phase 0 = 3 non-Faisal users selesaikan cycle (BLOCKING untuk sprint). Phase 1-3 mengikuti.
- DA C3 (moat rapuh) DICABUT; C1 dimodif (event-trigger bukan kalender); C2/C4/C5 dipertahankan.

**Gate G1 (deklarasi planning) — PO judge sendiri, tidak perlu tanya Faisal:**
- Sesi ini = PLANNING/MENTAL-MODEL. Output tidak masuk sprint sampai Phase 0 selesai (C5 berlaku untuk sprint execution, bukan perencanaan).

**PENDING CONFIRMATION dari Faisal (diajukan 2026-06-29, default 48 jam):**

Q1 (G2 Synthesizer): Apakah subscription Rp50rb/bulan DITOLAK, one-time per-cycle tetap sebagai satu-satunya mekanisme pembayaran?
- Default jika diam: DITOLAK (sesuai verdict Synthesizer)

Q2 (tambahan PO — lebih material dari G1): Apakah disclosure eksplisit di titik contribute WAJIB sebagai syarat non-negotiable — bukan cukup di privacy policy saja?
- Default jika diam: WAJIB (safer path untuk "user merasa in control atas datanya")

**Blind spot yang belum diputuskan (catat untuk masa depan):**
- Partnership data dengan riset akademis/media (bukan landlord/broker/developer) — zona abu-abu yang mungkin muncul. Perlu diputuskan sebelum ada tawaran datang.

**How to apply:** Sebelum membahas build untuk intel/data-collection mechanic, pastikan Q1 dan Q2 sudah dijawab. Sebelum Phase 1+, pastikan Phase 0 (3 cycle completions) sudah selesai.
