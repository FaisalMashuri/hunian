## T-001 — UX Researcher (PRIMARY)

Menghasilkan IA + spec 8 layar + design direction + component inventory.

**Inti:**
- Bottom nav 3 tab (Kandidat/Bandingkan/Pengaturan) + FAB. Gate login & onboarding_completed di middleware.
- 8 layar: Login landing → Onboarding 4-step → Input paste-first → Property Review ReviewRow ✓/⚠ (+ section input km manual) → Daftar Kandidat (status badge, deal breaker amber chip, empty state) → Skor+AI (skor kecil, bar dimensi, AI explanation conversational on-demand, tombol WhatsApp) → Compare (tabel horizontal-scroll sticky col + trade-off cards + dialog konfirmasi + exit survey sheet) → Settings (Sheet edit + recalculate).
- Design direction: Plus Jakarta Sans (atas Inter/Roboto), teal-700, stone-50, status 3-layer, deal breaker amber bukan merah, skor bar tipis supporting. Aksesibilitas: jangan andalkan warna saja; ARIA; focus ring; touch target 44px; input 16px cegah zoom iOS.
- Component inventory: 21 shadcn + 8 custom.

**Tradeoff yang dieksplisitkan:** navigasi (bottom nav vs hamburger), input (paste-first vs form-first), Compare layout (table+cards vs card-stack/carousel/tabel-saja), skor (kecil vs hero), deal breaker (amber vs merah), font (Jakarta Sans vs Inter).

**Risiko UX dicatat:** drop-off onboarding step 3-4; kelengkapan field "jarak km" (input di luar app); Compare underused bila hanya 2 kandidat; asumsi user bisa paste tanpa instruksi & trade-off framing beresonansi (belum divalidasi).
